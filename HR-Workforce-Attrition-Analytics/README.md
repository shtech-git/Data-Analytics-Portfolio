# HR Attrition Analytics Dashboard

## Dataset
**IBM HR Analytics Employee Attrition Dataset**

## Tools Used
- Excel
- SQL Server (SSMS)
- Python (Pandas & Plotly)
- Power BI

## Project Overview
This project analyzes employee attrition data to identify workforce trends, employee retention patterns, 
and the factors influencing attrition. Using Excel, SQL Server, Python (Pandas & Plotly), 
and Power BI, the project transforms HR data into interactive dashboards and business insights.

## Key Objectives
- Analyze employee attrition trends
- Compare attrition across departments and job roles
- Analyze employee demographics and workforce distribution
- Evaluate the impact of overtime, job satisfaction, and income on attrition
- Monitor salary and workforce metrics
- Build interactive dashboards for HR analysis
