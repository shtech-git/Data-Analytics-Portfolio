# Banking Operations & Customer Analytics Project

## Dataset
**Banking and Customer Transaction Data**

## Tools Used
- Excel
- SQL Server (SSMS)
- Power BI

## Project Overview
This project analyzes banking customer and transaction data to evaluate customer behavior, banking operations, and transaction trends. 
Using Excel, SQL Server, and Power BI, the project transforms raw data into interactive dashboards and business insights.

## Key Objectives
- Analyze customer demographics
- Analyze transaction trends
- Analyze investment patterns
- Compare account balances across customer segments
- Evaluate branch revenue and profit performance
- Build interactive dashboards for business analysis
